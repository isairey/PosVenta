---

## 📌 Descripción

Este proyecto es un **sistema de punto de venta (POS)** desarrollado con tecnologías modernas para la web:

- **Frontend:** Vue.js  
- **Backend:** PHP  
- **Base de datos:** MySQL  

El sistema permite gestionar ventas, cuentas, apartados y cotizaciones de manera eficiente, ideal para negocios pequeños y medianos.

---

## ⚙️ Características

- Registro de ventas de productos y servicios.  
- Gestión de cuentas y movimientos.  
- Registro de apartados y seguimiento de clientes.  
- Generación de cotizaciones y recibos.  
- Interfaz intuitiva y responsive.
